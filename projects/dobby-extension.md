# ⭐ Browser Extension: Chat Dobby 70B

> **Featured Work**

My Chrome extension for chatting with **Dobby-70B** (Sentient's language model) — now updated with a lot of exciting features.

### Features

* **Highlight anywhere** — select any text on any page, and Dobby will summarize it, explain it, guide you, or answer your questions.
* **Language selection** — choose the language Dobby responds in.
* **Personality modes** — pick a kind Dobby or an evil one with some spicy language.
* More features planned. 🚀
